# Позитивные тесты:

- 01 - все столбцы не подходят под условие;
- 02 - все столбцы убывающие;
- 03 - один столбец и одна строка;
- 04 - максимальная размерность;
- 05 - различные столбцы.

# Негативные тесты:

- 01 - пустой файл;
- 02 - пустая матрица;
